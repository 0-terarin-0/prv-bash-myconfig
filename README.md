# 環境自動設定スクリプト

## 仕様

自動で次の環境を設定します。

- zsh, curl, wget, git, gh
- Oh-My-ZSH,zsh-autosuggestions,rustup,Alacritty,zsh-theme


## 必須環境

- インターネット接続環境

## 利用方法

```bash
sh -c "$(wget https://raw.githubusercontent.com/0-terarin-0/prv-bash-myconfig/refs/heads/main/config.sh -O -)"
```

## ライブラリ

特にライブラリ等は使用していません。
